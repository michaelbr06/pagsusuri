# Project Rules: Pagsusuri

## Tech Stack

- Framework: No build
- Language: Plain JavaScript, HTML, and CSS
- Styling: CSS

## Overview

- This is a web app containing a series of guide questions to help Catholics "Examine" their conscience before going to Confession.
- Each question is an affirmative question such as "I have failed, I have wronged, etc."
- If the question is checked, it means that penitent has committed that particular sin.
- Upon submitting, the questions are then put into a list.

## Refactoring Preferences

- Favor readability over brevity.

## Terminal Protocol

- Go Live using the "Go Live" extension of VS Code.
